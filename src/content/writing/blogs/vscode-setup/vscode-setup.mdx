I've spent countless hours tweaking my VS Code setup. Here's what stuck.

## Essential Extensions

- **ESLint** - Code quality enforcer
- **Prettier** - Consistent formatting
- **GitLens** - Git superpowers
- **Error Lens** - Inline error highlighting
- **Auto Rename Tag** - HTML/JSX sanity

## Key Settings

```json
{
  "editor.fontSize": 14,
  "editor.fontFamily": "JetBrains Mono",
  "editor.formatOnSave": true,
  "editor.minimap.enabled": false,
  "workbench.colorTheme": "Vitesse Dark"
}
```

## Keyboard Shortcuts I Can't Live Without

| Shortcut | Action |
|----------|--------|
| `Ctrl+P` | Quick Open |
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+\`` | Toggle Terminal |
| `F2` | Rename Symbol |
| `Alt+Click` | Multi-cursor |

## Theme & Font

I use **Vitesse Dark** with **JetBrains Mono** font (with ligatures enabled). Clean, minimal, and easy on the eyes during long coding sessions.
